Dice Game

  • Regras do jogo
    • O Jogador lança um par de dados.
    • Se a soma das duas faces dos dados pode ser qualquer número entre 2 e 12.
    • Se o jogador lançar os dados e obter 7 ou 11 na primera jogada, o jogador vence.
    • Se o jogador lançar e obter 2, 3 ou 12, o jogador perde.
    • Para qualquer outro resultado (4, 5, 6, 8, 9 ou 10), um lançamento sequencial será solicitado.
